26 APIs fournies gouvernementales pour construire des services. Un accès unique aux API de l'État.

https://api.gouv.fr 

FranceConnect

Le Single Sign On (SSO) du gouvernement. 

https://franceconnect.gouv.fr 

Documentation geo.api.gouv.fr

Api Geo : les communes, départements et régions françaises. Interrogez les référentiels géographiques plus facilement. 

https://geo.api.gouv.fr 

APIs Pôle Emploi

APIs Pôle Emploi : tout un jeu d'APIs sur le marché du travail (annonces, entreprises, compétences...).

https://pole-emploi.io/data/api 

Le Taxi

Le Taxi : une plateforme qui permet de construire des applications de commande de taxis, utilisée notamment par G7. La plateforme fournit le service de localisation et de mise en relation, le registre des chauffeurs de taxis, mais pas l'application ni le business model. Les datas sont mutualisées et peuvent être exploitées anonymement par l'État : bel exemple de mise à disposition d'infrastructure gouvernementale compatible avec les initiatives locales, et de conservation de la ressource en données.

https://le.taxi

OpenFisca

Le système fiscal français codé en Python, accessible en REST ! Tout pour construire des applications de simulation.

https://fr.openfisca.org/

Base Adresse Nationale

Les adresses en France.

https://adresse.data.gouv.fr/

Entrepreneurs d'intérêt général

Entrepreneur.e d'Intérêt Général : intéressant programme de recrutement pour lancer des startups d'État. 

https://eig.etalab.gouv.fr 

beta.gouv.fr

L'incubateur des startups d'État. Parmi ces 66:

Locatio

Une plateforme pour rétablir la confiance entre bailleurs et locataires.

https://locatio.beta.gouv.fr 

API Particulier

API Particulier : une API pour fournir les justificatifs relatifs à un usager, automatiquement. A l'heure actuelle, avis d'imposition, adresse fiscale, quotient familial et composition familiale.

https://api.gouv.fr/les-api/api-particulier 

Démarches simplifiées

Une plateforme pour aider les administrations à dématérialiser et simplifier les démarches de leurs usagers.

https://www.demarches-simplifiees.fr 

Simulateur d'embauche

Un outil très simple et ergonomique pour comprendre le coût salarial.

https://mon-entreprise.urssaf.fr 

API Entreprise

Une API dédiée aux services publics fournissant des informations fiables sur les entreprises.

https://entreprise.api.gouv.fr 

PIX

Evaluation et certification des compétences numériques.

https://pix.fr
